[ONBOARDING] - Automatischer Setup-Assistent für einen schnellen Start

[STEP_01] Initialisierung

Das System prüft den Portablen-Modus. Es werden automatisch die Ordner `data/` und `logs/` erstellt, falls diese fehlen. Es ist keine manuelle Ordnerstruktur-Erstellung nötig.

[STEP_02] Smart Link Parser 🔗

Der Smart-Parser erkennt automatisch das Eingabeformat. Du musst dich nicht um die genaue URL kümmern:

# Einfacher Name
@username
# Vollständiger Link
https://www.tikleap.com/profile/0fficial404
  • Einfacher Name: `@username` wird zu TikLeap-URL.
  • TikTok Link: Automatische Extraktion des Usernames.
  • TikLeap Link: Übernahme des direkten Profil-Endpunkts.

[STEP_03] Team & Groups 👥

Ordne deine Accounts logisch zu. In der v2.0.9 kannst du Creatoren direkt beim Setup in Teams sortieren. Dies ermöglicht später das Filtern der Statistiken nach Gruppen.

  • Multi-Select: Wähle vordefinierte Gruppen oder erstelle neue Tags.
  • Struktur: Trenne z.B. "Privat", "Kunden" und "Test-Accounts".

[STEP_04] Auto-Pilot & Init 🚀

Der finale Schritt konfiguriert den Hintergrund-Dienst. Du entscheidest, wie oft der Crawler aktiv wird und ob er direkt minimiert im System-Tray starten soll.

  • Interval: Einstellen der Abfrage-Frequenz (Autopilot).
  • Tray-Start: Sofortiger Start als diskreter Hintergrundprozess.
  • Config Write: Erstellt final die settings.json und profiles.txt.
