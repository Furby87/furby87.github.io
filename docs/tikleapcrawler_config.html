[SETTINGS] - Globale Parameter & Performance-Konfiguration

🍪 cookie.txt

Erforderlich für den Zugriff auf geschützte TikLeap-Profile. Nutze eine Browser-Extension (z.B. "Get cookies.txt") im Netscape-Format.

# Netscape HTTP Cookie File
.tikleap.com	TRUE	/	TRUE	1735689600	session_id	xyz123...
.tikleap.com	TRUE	/	TRUE	1735689600	csrf_token	abc456...

⚙️ settings.json

Speichert deine globalen Präferenzen. Wird automatisch im Ordner /data verwaltet.

{
    "timeframe": "Letzte 30 Tage",
    "legend_size": "Legende: Mikro",
    "auto_interval": "Alle 6 Std",
    "use_proxies": true,
    "only_proxies": false,
    "view_mode": "Einzeln"
}

📱 user_agent.txt

Optional. Erlaube dem Crawler, sich als ein spezifisches Gerät auszugeben.

Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/124.0.0.0 Safari/537.36
