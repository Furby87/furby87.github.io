📊 Die 2x3 KPI Matrix

In der Version 2.0.8 haben wir die KPI-Karten restrukturiert, um eine logische Lesbarkeit zu gewährleisten. Die Matrix zeigt dir Ereignis-basierte Höchst- und Tiefstwerte:

  • Top-Row (Main Stats): Gesamter Umsatz, Durchschnitt und Monats-Trend.
  • Bottom-Row (Outlier Analytics): Der absolut beste Tag (Max) und der schwächste Tag (Min > $0).
  • Live-Calculations: Alle Werte werden in Echtzeit aus der `SQLite` Datenbank neu berechnet, wenn du einen Kanal auswählst.

🗄️ Smart Database Merge

Das neue merge_db.py Tool (v2.0.9) erlaubt das sichere Zusammenführen von mehreren Datenbanken:

  • UPSERT Logik: Bei Duplikaten wird automatisch der Datensatz mit dem neuesten Zeitstempel priorisiert.
  • Integrierte Sicherung: Vor jeder Operation wird automatisch eine .db.bak Datei erstellt.
  • Cross-Platform: Funktioniert nahtlos zwischen Desktop- und Termux-Instanzen.

Recovery & Modale: Über das ℹ️ Dashboard-Icon (About) kannst du jederzeit eine ältere Sicherung (.db.bak) direkt in der GUI wiederherstellen.

📥 Excel-Ready Exports

Beim Klick auf 📤 CSV Export wird automatisch eine Datei generiert, die ISO-konforme Datumsformate nutzt. Dies ermöglicht die direkte Weiterverarbeitung in **Microsoft Excel**, **Google Sheets** oder **Power BI**.

🔄 Account Migration-Tool

Ändert ein Creator seinen Link? Nutze das Migration-Tool, um die gesamte Historie in der Datenbank auf den neuen Namen umzuschreiben, ohne Datenverlust.
