💻 Portabilität

Der TikLeap Crawler ist vollständig Headless-kompatibel und läuft ressourcensparend auf Servern oder Android (Termux).

📜 Befehle

python main.py --help
  • --test-single: Prüft einen einzelnen Account isoliert.
  • --wizard: Startet das Setup im CLI-Dialog.
  • --test-schedule: Simuliert einen beschleunigten 24h-Run.
  • --headless: Führt den Crawler ohne GUI aus.

🗄️ merge_db.py

Zusammenführen von Datenbanken mit integriertem Backup-System und UPSERT-Logik. Perfekt für Migrationen oder Daten-Zusammenführungen.

python merge_db.py
  • Interaktiv: Ohne Argumente öffnet sich ein Datei-Dialog zur Auswahl der Datenbank.
  • Sicherheit: Erstellt vor dem Merge automatisch ein .db.bak Backup.
  • UPSERT Logik: Erkennt Duplikate und behält bei Überschneidungen den neuesten Zeitstempel.
