Aggregated views for creator agencies

👥 Team Dashboard Logic

Das Team-Feature (v2.0.5) erlaubt Agencies, hunderte Accounts in logischen Teams (`groups.json`) zu organisieren. Im Team-Modus aggregiert das Dashboard:

{
  "Team Alpha": [
    "creator_link_1",
    "creator_link_2"
  ]
}
  • Total Revenue: Die Live-Einnahmen aller Teammitglieder werden addiert.
  • Avg/Member: Durchschnittlicher Verdienst innerhalb der Gruppe.

🖊️ Integrated Group-Manager

Du musst keine JSON-Dateien manuell editieren. Über den Button 👥 Gruppen verwalten öffnet sich ein sicherer Editor mit Echtzeit-Validierung.
