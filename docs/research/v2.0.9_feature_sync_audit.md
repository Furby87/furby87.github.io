# Research: v2.0.9 Feature Audit

**Date**: 2026-04-18
**Source**: Analysis of Crawler README.md and CHANGELOG.md

## Identifizierte v2.0.9 Key-Features

Aus dem Audit der Source-Repository wurden folgende Features extrahiert, die in der Dokumentation fehlten oder veraltet waren:

### 1. UI & Productivity

- **ABC Sort**: Alphabetische Sortierung der Sidebar.
- **Reload Config**: Hot-Reload von `proxies.txt` und `profiles.txt` ohne Neustart.
- **Exit Logic**: Dialog beim Beenden (Wahl zwischen Tray-Minimierung und echtem Exit).
- **Sidebar Hierarchy**: Gruppierung in "Tracking", "Monitor" und "Tools".

### 2. Setup Wizard

- **Teams & Groups**: Möglichkeit, Creatoren direkt beim Onboarding Gruppen zuzuweisen.
- **Auto-Pilot**: Integration der Intervall-Einstellungen direkt in den Wizard.

### 3. Data & Proxy

- **Echtzeit-Fortschritt**: Anzeige des Crawler-Status (z.B. `⏳ 45/901`) in der GUI.
- **Database Recovery**: Wiederherstellung von `.db.bak` Sicherungen direkt über das Dashboard (About-Modale).
